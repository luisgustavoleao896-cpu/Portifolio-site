Aprendizado: Orquestração de stack completo, deploy com variáveis de ambiente e documentação de API com OpenAPI/Swagger.

Sistema de Login Seguro
Node.js • bcrypt • JWT

Módulo de autenticação completo com hash de senhas, tokens de sessão, validação de formulário, proteção contra brute force e recuperação de senha.

Problema resolvido: Implementar autenticação segura em aplicações web, protegendo dados sensíveis de usuários contra ataques comuns como credential stuffing e session hijacking.
Node.js Express bcrypt.js JWT dotenv
Decisões técnicas:
  • bcrypt com salt rounds = 12 para equilíbrio segurança/performance
  • Refresh tokens com rotação para sessões persistentes seguras
  • Rate limiting por IP para prevenir ataques de força bruta

Aprendizado: Segurança em aplicações web, hash de senhas com salt, middleware de autenticação, variáveis de ambiente e boas práticas de OWASP.
